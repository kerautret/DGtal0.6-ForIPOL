Detailed Description

template<typename T>
struct DGtal::CPointEmbedder< T >

Aim: A point embedder is a mapping from digital points to Euclidean points. It adds inner types to functor.

Description of concept 'CPointEmbedder'

Refinement of

  • CUnaryFunctor<T, T::Point, T::RealPoint>

Associated types :

  • Space: the digital space
  • Point: same as Space::Point
  • Argument: type of the argument, same as Space::Point
  • Value: type of value, same as Space::RealPoint

Notation

  • X : A type that is a model of CPointEmbedder
  • x : object of type X
  • p : object of type Point
